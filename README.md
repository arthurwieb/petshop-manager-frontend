
## Getting Started

First, run the development server:

```bash
docker build -t pet-manager-frontend .
docker run -p 3000:3000 pet-manager-frontend:latest

OR

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
